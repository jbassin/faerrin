## Overview
Stakeholders raised a concern, but I'm not sure if we'd have addressed it yet: since the transcripts are machine-generated, it's possible that some of the phrasing is incorrectly transcribed. For instance, the "Roundhat" gang sometimes appears in the transcripts as "Roundhack" gang. Also, some characters are called by different names, like "Gin-Soaked Rag" is also "Ginny". I'm not sure if it would have been addressed yet in the code or if there are plans to address it later.
